# Opinify

Opinify is your platform to voice your thoughts, share your ideas, and connect with a diverse community. Speak freely, share your unique perspective, and foster meaningful connections through vibrant discussions. It's a dynamic space where opinions thrive.

## Technologies Used

Opinify is built using the following technologies:

- Next.js 13
- React.js
- TypeScript
- Tailwind CSS
- Prisma
- Redis
- PostgreSQL

## Features and Flow

Opinify provides a range of features and a seamless user flow:

- **Explore Without Account**: Users can explore the app without creating an account.
- **User Registration**: Users can create an account by signing up.
- **Community Management**: Once registered, users can join, leave, or create communities on Opinify.
- **Participation in Discussions**: Users can take part in any discussion thread on Opinify.
- **Creation of Posts and Discussions**: Registered users can create new posts or discussions within the communities they've joined.
- **Customized Feed**: Opinify provides a customized feed for signed-up users based on their past preferences, while non-signed-up users receive an uncategorized feed.
- **Voting System**: Users can upvote and downvote posts and comments within discussion threads.
- **Infinite Scrolling**: The app utilizes infinite scrolling for a seamless user experience.
- **Search Functionality**: Users can search discussions, posts, or communities using the search bar.

## Getting Started

To get started with Opinify, follow these steps:

1. Clone the repository: `git clone https://github.com/yourusername/opinify.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Access Opinify in your browser at `http://localhost:3000`
